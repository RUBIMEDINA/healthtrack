# HealthTrack 🏥

## 📝 Descripción
HealthTrack es una API REST para gestión de citas médicas con enfoque en privacidad de datos.

## 🚀 Tecnologías
- Node.js + Express
- MongoDB Atlas
- JWT para autenticación
- Swagger para documentación

## ⚙️ Instalación

```bash
# Instalar dependencias
npm install

# Configurar variables de entorno (editar .env)
cp .env.template .env

# Ejecutar en desarrollo
npm run dev

# Ejecutar en producción
npm start
